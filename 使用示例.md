# Rust Excel 拆分工具 - 使用示例

## 基本用法

### 1. 查看帮助

```bash
excel-splitter.exe --help
```

输出：
```
High-performance Excel file splitter with format preservation

Usage: excel-splitter.exe [OPTIONS] --input <INPUT> --sheets <SHEETS> --column <COLUMN>

Options:
  -i, --input <INPUT>              Input Excel file
  -s, --sheets <SHEETS>            Sheets to split (comma-separated)
  -r, --header-row <HEADER_ROW>    Header row number (1-based) [default: 1]
  -c, --column <COLUMN>            Column name to split by
  -p, --public-sheets <PUBLIC_SHEETS>  Public sheets to copy (comma-separated)
  -o, --output <OUTPUT>            Output directory [default: split_output]
  -t, --template <TEMPLATE>        Filename template [default: {base}_{group}.xlsx]
      --empty-handling <EMPTY_HANDLING>  Empty value handling [default: empty_group]
  -v, --verbose                    Enable verbose logging
  -h, --help                       Print help
  -V, --version                    Print version
```

## 使用示例

### 示例 1：基本拆分

**场景：** 按"区域"列拆分 Sheet1

```bash
excel-splitter.exe -i sales_data.xlsx -s "Sheet1" -c "区域"
```

**输出：**
```
split_output/
├── sales_data_华东.xlsx
├── sales_data_华北.xlsx
├── sales_data_华南.xlsx
└── sales_data_西南.xlsx
```

---

### 示例 2：多 Sheet 拆分

**场景：** 银行数据，4 个业务 Sheet 按"机构"拆分

```bash
excel-splitter.exe \
  -i banking_data.xlsx \
  -s "存款业务,贷款业务,理财业务,信用卡业务" \
  -r 2 \
  -c "所属机构" \
  -p "参数表" \
  -o ./output
```

**输入文件结构：**
```
banking_data.xlsx
├── 存款业务 (需要拆分)
├── 贷款业务 (需要拆分)
├── 理财业务 (需要拆分)
├── 信用卡业务 (需要拆分)
└── 参数表 (公共 Sheet)
```

**输出：**
```
output/
├── banking_data_北京分行.xlsx
│   ├── 存款业务 (仅北京分行数据)
│   ├── 贷款业务 (仅北京分行数据)
│   ├── 理财业务 (仅北京分行数据)
│   ├── 信用卡业务 (仅北京分行数据)
│   └── 参数表 (完整复制)
├── banking_data_上海分行.xlsx
│   ├── 存款业务 (仅上海分行数据)
│   ├── 贷款业务 (仅上海分行数据)
│   ├── 理财业务 (仅上海分行数据)
│   ├── 信用卡业务 (仅上海分行数据)
│   └── 参数表 (完整复制)
└── banking_data_广州分行.xlsx
    ├── 存款业务 (仅广州分行数据)
    ├── 贷款业务 (仅广州分行数据)
    ├── 理财业务 (仅广州分行数据)
    ├── 信用卡业务 (仅广州分行数据)
    └── 参数表 (完整复制)
```

### 示例 3：自定义文件名

**场景：** 使用自定义文件名模板

```bash
excel-splitter.exe \
  -i report.xlsx \
  -s "Sheet1" \
  -c "部门" \
  -t "{base}_{group}_Report_2024.xlsx" \
  -o ./reports

**输出：**
```
reports/
├── report_销售部_Report_2024.xlsx
├── report_技术部_Report_2024.xlsx
└── report_人事部_Report_2024.xlsx

### 示例 4：处理空值

**场景：** 跳过空值行

```bash
excel-splitter.exe \
  -i data.xlsx \
  -s "Sheet1" \
  -c "分类" \
  --empty-handling skip
```

**场景：** 遇到空值停止并报错

```bash
excel-splitter.exe \
  -i data.xlsx \
  -s "Sheet1" \
  -c "分类" \
  --empty-handling stop
```

## 批量处理脚本

### Windows 批处理

```batch
@echo off
set INPUT_DIR=./input
set OUTPUT_DIR=./output

for %%f in (%INPUT_DIR%\*.xlsx) do (
    echo Processing: %%f
    excel-splitter.exe \
        -i "%%f" \
        -s "Sheet1" \
        -c "分类" \
        -o "%OUTPUT_DIR%/%%~nf"
)

echo All files processed!
pause
```

### PowerShell 脚本

```powershell
$inputDir = "./input"
$outputDir = "./output"

Get-ChildItem $inputDir -Filter "*.xlsx" | ForEach-Object {
    Write-Host "Processing: $($_.Name)"
    
    $outputSubDir = Join-Path $outputDir $_.BaseName
    
    & ./excel-splitter.exe `
        -i $_.FullName `
        -s "Sheet1" `
        -c "分类" `
        -o $outputSubDir
}

Write-Host "All files processed!"
```

## 常见问题

### Q: 如何查看 Excel 中的 Sheet 名称？

**A:** 运行程序时会自动显示：
```bash
excel-splitter.exe -i data.xlsx -s "test" -c "test"
# 会显示：Available sheets: Sheet1, Sheet2, Sheet3
```

### Q: 列名包含空格怎么办？

**A:** 使用引号包裹：
```bash
excel-splitter.exe -i data.xlsx -s "Sheet1" -c "所属 机构"
```

### Q: 文件名包含非法字符怎么办？

**A:** 程序会自动处理，将非法字符替换为下划线：
```
输入：北京/分公司
输出：北京_分公司.xlsx
```

### Q: 如何处理大量数据？

**A:** Rust 版本内存效率高，可以处理大文件：
```bash
# 处理 100MB+ 文件
excel-splitter.exe -i large_file.xlsx -s "Sheet1" -c "分类" -v
```

## 性能优化建议

1. **使用 SSD**：输出目录放在 SSD 上可大幅提升写入速度
2. **关闭杀毒软件**：实时扫描会影响文件写入速度
3. **分批处理**：超大文件可以分批处理
4. **使用 Release 版本**：确保使用 `--release` 编译的版本
