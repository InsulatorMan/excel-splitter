# Tauri GUI 版本 - 快速编译指南

## 项目结构

```
rust_excel_splitter/
├── src-tauri/              # Tauri 后端代码
│   ├── Cargo.toml          # Rust 依赖配置
│   ├── tauri.conf.json     # Tauri 配置
│   ├── build.rs            # 构建脚本
│   └── src/
│       ├── main.rs         # 主程序
│       └── format.rs       # 格式处理
├── ui/                     # 前端界面
│   └── index.html          # GUI 页面
├── build.bat               # Windows 一键编译脚本
└── 快速编译指南.md          # 本文件
```

## 在另一台电脑编译步骤

### 1. 安装 Rust

**Windows:**
1. 下载 https://win.rustup.rs/x86_64
2. 运行 `rustup-init.exe`
3. 选择默认选项（按回车）
4. 等待安装完成
5. 重启命令行

**验证:**
```bash
rustc --version
cargo --version
```

### 2. 安装系统依赖

**Windows 必需:**
- 安装 Visual Studio 2022 或 Build Tools
- 下载: https://visualstudio.microsoft.com/downloads/
- 勾选 "使用 C++ 的桌面开发"

**macOS 必需:**
```bash
xcode-select --install
```

**Linux 必需 (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install libwebkit2gtk-4.0-dev build-essential curl wget libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev
```

### 3. 复制项目文件

将以下文件复制到目标电脑：

```
rust_excel_splitter/
├── src-tauri/
│   ├── Cargo.toml
│   ├── tauri.conf.json
│   ├── build.rs
│   └── src/
│       ├── main.rs
│       └── format.rs
└── ui/
    └── index.html
```

### 4. 编译

**方式 1: 使用一键编译脚本 (Windows)**
```bash
cd rust_excel_splitter
build.bat
```

**方式 2: 手动编译**
```bash
cd rust_excel_splitter/src-tauri

# 安装 Tauri CLI
cargo install tauri-cli

# 编译发布版本
cargo tauri build
```

### 5. 获取编译结果

编译完成后，文件位置：

**Windows:**
- 可执行文件: `src-tauri/target/release/excel-splitter.exe`
- 安装包: `src-tauri/target/release/bundle/msi/*.msi`

**macOS:**
- 应用: `src-tauri/target/release/bundle/macos/*.app`
- 安装包: `src-tauri/target/release/bundle/dmg/*.dmg`

**Linux:**
- 可执行文件: `src-tauri/target/release/excel-splitter`
- DEB 包: `src-tauri/target/release/bundle/deb/*.deb`

## 常见问题

### Q: 编译失败，提示找不到 WebView2
**A:** Windows 10/11 已内置 WebView2。如果是 Windows 7/8，需要下载安装。

### Q: 编译失败，提示缺少 MSVC
**A:** 确保安装了 Visual Studio 的 "使用 C++ 的桌面开发" 工作负载。

### Q: 编译后的文件很大
**A:** 确保使用 `--release` 模式编译，已配置优化选项。

### Q: 如何在没有 Rust 环境的电脑运行？
**A:** 编译后的 `.exe` 文件是纯二进制，可以直接复制到其他 Windows 电脑运行，无需安装 Rust。

## 分发说明

编译完成后，您可以：

1. **直接分发 EXE 文件**
   - 复制 `excel-splitter.exe`
   - 其他电脑直接运行

2. **创建安装包 (MSI)**
   - 使用 `cargo tauri build` 自动生成
   - 用户双击安装

3. **压缩分发**
   ```bash
   # 创建 ZIP 压缩包
   7z a excel-splitter.zip excel-splitter.exe README.md
   ```

## 性能优化

编译配置已在 `Cargo.toml` 中优化：

```toml
[profile.release]
opt-level = 3        # 最高优化
lto = true           # 链接时优化
codegen-units = 1    # 单代码生成单元
panic = "abort"      # 简化 panic
strip = true         # 移除符号
```
