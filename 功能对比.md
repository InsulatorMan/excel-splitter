## 使用方式对比

### Python 版本

```bash
# GUI 模式
python excel_splitter_gui.py

# 命令行模式
python excel_splitter.py input.xlsx --sheets "Sheet1" --column "机构"
```

### Rust 版本

```bash
# 命令行模式（当前唯一方式）
excel-splitter.exe -i input.xlsx -s "Sheet1" -c "机构"

# 多 Sheet 拆分
excel-splitter.exe -i input.xlsx -s "Sheet1,Sheet2" -c "机构" -p "参数表"
```

## 适用场景建议

### 选择 Python 版本如果：
- ✅ 需要图形界面
- ✅ 需要完整格式保留
- ✅ 开发时间紧迫
- ✅ 团队熟悉 Python
- ✅ 需要快速迭代

### 选择 Rust 版本如果：
- ✅ 追求极致性能
- ✅ 需要小体积分发
- ✅ 处理大文件
- ✅ 无环境依赖要求
- ✅ 长期稳定运行
