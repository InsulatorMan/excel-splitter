# 项目结构说明

## 目录结构

```
rust_excel_splitter/
├── .github/
│   └── workflows/
│       └── build.yml           # GitHub Actions 自动编译配置
├── src-tauri/                   # Tauri 项目目录
│   ├── Cargo.toml              # Rust 依赖配置
│   ├── tauri.conf.json         # Tauri 配置
│   ├── build.rs                # 构建脚本
│   └── src/
│       ├── main.rs             # 主程序
│       └── format.rs           # 格式处理
├── ui/
│   └── index.html              # GUI 界面
├── .gitignore                  # Git 忽略文件
├── Cargo.toml                  # 根项目配置（可选）
├── README.md                   # 项目说明
├── build.bat                   # Windows 本地编译脚本
└── ...                         # 其他文档
```

## 关键文件说明

### GitHub Actions 配置

`.github/workflows/build.yml`
- 自动在 Windows/Linux/macOS 编译
- 创建 Release 并上传文件

### Tauri 配置

`src-tauri/tauri.conf.json`
- 窗口大小、标题
- 权限设置
- 打包配置

### Rust 主程序

`src-tauri/src/main.rs`
- Excel 读取和拆分逻辑
- Tauri 命令接口

### GUI 界面

`ui/index.html`
- 现代化 Web 界面
- 文件选择、配置、预览

## 编译输出

GitHub Actions 编译后会生成：

| 平台 | 文件类型 | 说明 |
|------|----------|------|
| Windows | `.exe` | 可执行文件 |
| Windows | `.msi` | 安装包 |
| Linux | `.deb` | Debian 包 |
| Linux | `.AppImage` | 便携包 |
| macOS | `.app` | 应用程序 |
| macOS | `.dmg` | 安装镜像 |
